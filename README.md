# projeto-final
Projeto final da UC - Laboratório de Projeto II do aluno André Gomes com o número 3230146

## Descrição
O projeto final tem como objetivo a criação de um site de uma empresa fictional criada pelo aluno com diversos projetos criados pela mesma (criados pelo aluno, mas como se o aluno fosse a empresa).

Os trabalhos desenvolvidos na linguagem **Processing** devem estar organizados na página `processing.html`, sendo apresentados nos respetivos **canvas**.

As páginas `index.html`, `sobre-nos.html`, `projetos.html` e `contactos.html` são de conteúdo livre, sendo definidas pelos autores do projeto.

---

## Objetivos
- Desenvolver um website profissional tendo em conta obstáculos comerciais (site de empresa);
- Organizar e apresentar trabalhos académicos;
- Aplicar conhecimentos de HTML, CSS e JavaScript;
- Integrar projetos desenvolvidos em Processing;
- Criar uma identidade gráfica para uma empresa fictional.

## Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Processing  
- Adobe Illustrator 2026 

---

## Páginas do Website

- **index.html** → Página inicial  
- **sobre-mim.html** → Informação sobre a empresa
- **projetos.html** → Apresentação dos trabalhos desenvolvidos  
- **contactos.html** → Contactos do autor  
- **processing.html** → Projetos desenvolvidos em Processing (com canvas)
